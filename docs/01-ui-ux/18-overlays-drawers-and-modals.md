# Overlays, Drawers, and Modals

## Modal

Gunakan untuk keputusan singkat dan berisiko. Jangan menaruh form panjang checkout atau product editor di modal.

## Drawer

Desktop detail/audit/filter dapat memakai side drawer. Mobile filter, address picker, variant picker, dan action menu memakai bottom sheet.

## Rules

- Focus trap.
- Initial focus masuk ke heading atau field pertama.
- Focus kembali ke trigger setelah ditutup.
- Background scroll terkunci.
- Escape menutup kecuali proses kritis sedang submit.
- Destructive confirmation menyebut item dan dampak.
