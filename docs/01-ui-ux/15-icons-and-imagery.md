# Icons and Imagery

## Icon

- Gunakan satu icon set lintas aplikasi.
- Ukuran umum 16, 20, 24px.
- Icon-only button minimum 44×44px pada mobile.
- Icon dekoratif memakai `aria-hidden=true`.

## Product image

- Aspect ratio product card konsisten.
- Gunakan responsive image, lazy loading, blur/skeleton placeholder.
- Hindari crop yang menghilangkan informasi produk.
- Varian dapat mengganti galeri tanpa layout shift besar.

## Empty state illustration

Boleh digunakan tetapi tidak menggantikan headline, penjelasan, dan CTA.
