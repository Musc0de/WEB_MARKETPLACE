# Responsive Data Tables

## Desktop

Gunakan table untuk order, invoice, inventory, customer, dan admin data. Header dapat sticky, column dapat hide/reorder berdasarkan prioritas.

## Mobile

Jangan memaksa horizontal scroll untuk semua data. Ubah row menjadi card berisi:

- identifier
- status
- nominal/tanggal
- primary action
- expandable secondary detail

Horizontal scroll hanya untuk comparison matrix atau data yang tidak dapat diubah menjadi card tanpa kehilangan makna.

## Accessibility

Table semantic tetap digunakan pada desktop. Sorting state diumumkan. Row action menu dapat dijangkau keyboard.
