# Feedback and Overlay Components

Feedback:

- Inline alert.
- Banner.
- Gooey Toast wrapper.
- Progress/status panel.
- Notification badge.

Overlay:

- Dialog.
- AlertDialog.
- Drawer.
- BottomSheet.
- Popover.
- Tooltip.

Pilih berdasarkan durasi, risiko, dan kebutuhan user untuk merespons. Toast adalah non-blocking;
confirmation adalah blocking.
