# Responsive Components

Komponen shared menerima layout melalui CSS/container query, bukan mengecek user-agent.

Komponen:

- `ResponsiveStack`
- `ResponsiveGrid`
- `DesktopOnly`/`MobileOnly` hanya bila composition benar-benar berbeda
- `AdaptiveTable`
- `FilterBar/FilterSheet`
- `StickyActionBar`
- `SafeAreaContainer`
- `ResponsiveGooeyToaster`

Business data dan action handler tetap satu sumber.
