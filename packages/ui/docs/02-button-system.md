# Button System

## Variants

Primary, secondary, outline, ghost, destructive, link, icon-only.

## Sizes

Small 32–36px, medium 40–44px, large 48–52px. Mobile primary action minimum 44px.

## States

Default, hover, pressed, focus-visible, disabled, loading. Loading tidak mengubah lebar secara
drastis.

## Action feedback

Button menjalankan action; Gooey Toast menampilkan hasil. Gooey Toast bukan button library utama.
Action button di dalam toast digunakan hanya untuk contextual follow-up seperti Undo/View/Retry.

## Rules

- Satu primary action per area.
- Icon-only wajib label.
- Destructive action tidak menggunakan success styling sebelum server confirmation.
- Link navigation tidak dibuat sebagai submit button.
