# Admin Forms, Tables, and Bulk Actions

## Tables

- Column priority and saved views.
- Sticky header.
- Server-side sort/filter/pagination.
- Row selection with clear selected count.
- Bulk toolbar only after selection.
- Export is asynchronous for large data.

## Forms

- Page-level forms for complex entities.
- Drawer for small contextual edit.
- Error summary and field errors.
- Unsaved changes guard.
- Version/conflict handling when record changed elsewhere.

## Bulk actions

1. Show exact scope/count.
2. Validate permission and eligibility server-side.
3. Require reason for risky action.
4. Show progress/result summary.
5. Allow download/view failed rows.
6. Audit every result.

Gooey Toast may show batch start and completion, but not replace the result report.
