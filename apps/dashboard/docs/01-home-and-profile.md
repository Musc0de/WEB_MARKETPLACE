# Home dan Profile

## Home

URL: `dashboard.starsuperscare.net/home`

Tampilkan:

- Pesanan aktif.
- Pesanan terakhir.
- Shipment berjalan.
- Total pembelian.
- Pembayaran belum selesai.
- Notifikasi terbaru.
- Quick action ke orders, invoice, cart, dan support.
- Rekomendasi dan beli lagi.

Setiap widget mempunyai loading skeleton, empty state, error state, dan link detail.

## Profile

URL: `dashboard.starsuperscare.net/profile`

Field:

- Foto profil.
- Nama lengkap.
- Username.
- Email dan status verifikasi.
- Nomor telepon.
- Tanggal bergabung.

Password asli tidak pernah ditampilkan.

## Security

URL: `dashboard.starsuperscare.net/settings/security`

- Ubah password.
- Daftar session/device.
- Logout semua perangkat.
- Login history.
- 2FA sebagai fase lanjutan.

## Toast

- `Profil berhasil diperbarui`.
- `Password berhasil diubah`.
- `Semua perangkat lain telah logout`.
